INSTRUCTIONS

Full steps from RTF export in Football Manager to in-game faces.

1. Download the facepack and install filters/views

  1. Download the Facepack and the Mapper app from the Download section above. The facepack includes a filters folder and a views folder.
  2. Extract the facepack into your FM graphics folder, e.g. Documents → Sports Interactive → Football Manager 2024 → graphics. Use a folder name like Wonderkid2Legend Facepack.
  3. Copy everything in the facepack’s filters folder into Documents → Sports Interactive → Football Manager 2024 → filters.
  4. Copy everything in the facepack’s views folder into Documents → Sports Interactive → Football Manager 2024 → views.

2. In-game: create a new unemployed manager

Create a new unemployed manager in your save and do all of the steps below as that manager, so “Players in Range” and the export are from a global scope.

3. Scouting → Players in Range

  1. Go to Scouting → Players in Range.
  2. View: Click the view dropdown (e.g. “Overview”) → Custom → Import View → choose SCRIPT FACES player search.fmf (from the views you installed). Select that view (it may show as “NewGAN” or similar).
  3. Filter: New Search → cog (bottom left) → Manage Filters → Import → select is newgen search filter → Load. Back in the search popup: cog → select is newgen search filter.
  4. Optional: in the filter area, use the Exclude dropdown and uncheck your team so your club’s regens aren’t in the list.
  5. You should see UID, Nationality, 2nd Nationality, Name, etc. Check that you’re looking at regens only.

4. Export the list (your RTF file)

  1. Click one row, then Ctrl+A to select all. (If the list is huge, wait until selection is done.)
  2. Ctrl+P (Print). In the dialog, choose Text File and save the file (e.g. players.rtf). That’s your Players RTF.
  3. Optional: repeat from a Staff view (with the right view/filter) to get Staff RTF. Staff mapping only includes regen staff.

5. Run the mapper

  1. Open the Mapper app.
  2. Select your Facepack folder (e.g. Wonderkid2Legend Facepack inside your FM graphics folder).
  3. Select your Players RTF file. Optional: select Staff RTF if you exported staff.
  4. Click Run. The app writes config.xml in the facepack folder.

6. Reload in Football Manager

Go to Preferences → Reload Skin (or restart the game). Your faces should now appear in-game.
